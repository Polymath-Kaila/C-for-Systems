# C for Systems and Security

Deep C learning repo focused on systems programming, UNIX internals, memory, processes, threads, sockets, ELF, debugging, and defensive security tooling.

## Core Direction

This is not a beginner C repo. The goal is to understand C as a systems language:

- Memory and object representation
- Pointers, arrays, structs, alignment
- Compilation, linking, loading
- File descriptors and syscalls
- Processes, signals, pipes, threads
- Sockets and protocol parsing
- Debugging with GDB, sanitizers, Valgrind
- Secure C patterns and vulnerability prevention

## Main References

- The C Programming Language, Kernighan and Ritchie
- Expert C Programming, Peter van der Linden

## Workflow

Each topic should include:

1. Concept
2. Memory view
3. Compiler view
4. Kernel or OS view
5. Security relevance
6. Minimal C example
7. Debugging commands
8. Notes from the books
